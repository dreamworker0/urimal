# sw-rewriting-playbook.md — 사회복지 문서 윤문 처방 레시피

> `korean-style-rewriter` 에이전트가 `sw-tell-taxonomy.md`의 finding을 받아
> 실제 수정을 수행할 때 참조하는 처방 레시피입니다.  
> 패턴ID는 `sw-tell-taxonomy.md`와 1:1 대응합니다.

---

## 처방 원칙

1. **가장 자연스러운 대체어 하나를 선택한다.** 여러 후보 중 문맥·장르에 맞는 것 하나.
2. **변경 최소 원칙**: finding이 있는 span만 수정. 주변 문장은 손대지 않는다.
3. **사회복지 문서 장르 유지**: 계획서·보고서의 격식과 정중함을 유지한다.

---

## SW-01: 호응·짝 오류

### SW-01-A 주어-술어 불일치
- 주어를 먼저 파악하고, 술어가 그 주어에 어울리는지 확인한다.
- 주어가 행위자인데 술어가 상태 서술이면 → 행위 동사로 교체.
- 주어가 여럿인데 술어가 하나면 → 각각에 맞는 술어 병렬 구성.

### SW-01-B 부사-동사 불일치
- "일주일간"처럼 기간 부사 뒤에 "시작하다"가 오면 → "일주일간의 ~을 시작하다"로 명사구화.
- "수시로"+"전하다" 조합은 자연스럽지 않으면 → "수시로 제공하다"로 교체.

### SW-01-C ~부터 ~까지 짝 누락
- "~까지"만 있으면 앞에 "~부터"를 복원하거나, "~까지"를 삭제하고 나열로 교체.

### SW-01-D 높임말 호응 불일치
- 주체가 높임 대상일 때: 조사 → "께서", 서술어 → "~시(었)습니다".
- 객체가 높임 대상일 때: "이야기를" → "말씀을", "주다" → "드리다".
- 압존법 적용: 청자가 더 높을 때 중간 인물 공대 생략.

### SW-01-E 주어 누락으로 행위자 혼동
- 주어를 명시하거나, 두 문장을 분리하여 각 주어를 드러낸다.

## SW-02: 논리 오류

### SW-02-A 역접 오류
- `~지만` 앞뒤가 실제로 대립·반전 관계인지 확인.
- 대립이 아니면 → `~라서`, `~이기에`, `~덕분에` 등 인과·이유 접속으로 교체.

### SW-02-B 인과 오류
- "예상하기 어렵지만 바라봅니다" 류 → "예상하기 어려우니 바라봅니다"

### SW-02-C·D 시제·맥락 모순
- 날짜·시점이 문맥과 맞지 않으면 → 오류 지적 후 롤백 없이 hold_and_report.

---

## SW-03: 시제·태·구조 통일

### SW-03-A 했습니다/하였습니다 혼용
- 문서 첫 등장 형태를 기준으로 전체 통일. 기본은 "했습니다"로.

### SW-03-B 높임말 통일
- 특정 대상에게 쓰는 높임 수준을 문서 전체에서 통일.
- "어르신들께서 담당합니다" → "어르신들께서 담당하십니다" (또는 전체 낮춰 통일).

### SW-03-C 나열 구조 불균형
- 명사절 나열이면 모두 명사절로, 동사절 나열이면 모두 동사절로.
- "어떤 음식을 만들면 좋을지, 언제 할지" → "무엇을 만들지, 언제 만들지"

---

## SW-04: 관용구 오남용

### SW-04-A `관계하다` 대체어
문맥에 따라 선택:
- 일상적 왕래 → "어울리다", "왕래하다"
- 연락·소통 → "소통하다", "연락하다"
- 신뢰 관계 → "함께하다"

### SW-04-B `고민하다` 대체어
- 깊이 생각함 → "숙고하다", "숙려하다"
- 검토·분석 → "검토하다", "살피다"
- 고뇌·걱정 → "염려하다", "걱정하다"
- 맥락에 따라: "궁리하다", "연구하다", "성찰하다"

### SW-04-C `이야기 나누다` 대체어
- 공식 회의 → "논의하다", "의논하다", "회의하다"
- 비공식 대화 → "대화하다", "담소하다"
- 상담 → "상담하다", "면담하다"

### SW-04-D `물론` 오용
- 앞 내용이 당연하지 않으면 → 삭제 또는 "그런데", "다만"으로 교체.

### SW-04-E `가지다`·`갖다` 명사화 오용
| 원문 | 수정 |
|---|---|
| 시간을 가졌습니다 | ~했습니다 |
| 모임을 갖는다 | 모인다 |
| 평가회를 가졌다 | 평가했다 / 열었다 |
| 효력을 가진다 | 효력이 있다 |

### SW-04-F `대하여`·`대해서` 과잉
- "~에 대해서"를 → 조사 "~에", "~을", "~는"으로 간결하게.
- "필요성에 대해서 원하고" → "필요성은 원하고" / "필요성을 원하고"

### SW-04-G `에 있어서` 과잉
- "~에 있어서" → "~에서", "~할 때", "~에"

## SW-05: `해 주다` 과잉

### SW-05-A 당사자 행위에 `~해 주다`
- 당사자가 스스로 한 일에는 "~해 주다" 제거.
- "참여해 주셨습니다" → "참여하셨습니다"
- "신청해 주신" → "신청하신"

### SW-05-B 행위자 혼동
- 누가 한 일인지 불분명하면 → 주어를 명시하고 "~해 주다" 제거.

---

## SW-06: 사업화 문체

### SW-06-A 명사+활동/하다 → 동사화
| 원문 | 수정 |
|---|---|
| 독서활동을 했다 | 책을 읽었다 |
| 요리활동을 했다 | 요리했다 |
| 김장활동을 했다 | 김장했다 |
| 만들기를 했다 | 만들었다 |
| 그리기를 했다 | 그렸다 |
| 모임을 했다 | 모였다 |

### SW-06-B `나눔` 명사화 해체
- "사례 나눔을 하면" → "사례를 나누면"
- "배움 나눔을 하며" → "배움을 나누며"

### SW-06-C 용언 쪼개기 복원
| 원문 | 수정 |
|---|---|
| 접수를 받고 있습니다 | 접수하고 있습니다 |
| 신청을 하셔야 합니다 | 신청하셔야 합니다 |
| 걱정이 되다 | 걱정되다 |
| 이용이 되고 있습니다 | 이용하고 있습니다 |
| 가입을 했습니다 | 가입했습니다 |

## SW-07: 과거완료 남용

### SW-07-A·B 이중 과거 → 단순 과거
- `했었~` → `했~`
- "계획했었는데" → "계획했는데"
- "참여하셨었는데" → "참여하셨는데"

---

## SW-08: 모호한 표현

### SW-08-A `~것 같아요` → 직접 표현
- 자기 감정·판단 → "좋아요", "어렵습니다", "옳습니다"

### SW-08-B `부분` → 구체적 단어
- "걱정되는 부분" → "걱정되는 점" / "걱정"
- "아쉬운 부분" → "아쉬운 점" / "아쉬움"

### SW-08-C `지점` → 구체적 단어
- "함께할 수 있는 지점" → "함께할 수 있는 일" / "접점"

### SW-08-D·E `~도록 하겠습니다` / `~기로 하겠습니다`
- "가도록 하겠습니다" → "가겠습니다"
- "마치기로 하겠습니다" → "마치겠습니다"

### SW-08-F `~라는 생각이 들었습니다`
- 생각 내용을 직접 진술: "~라고 생각했습니다" 또는 "~입니다"

### SW-08-G `~할 수 있는 시간이었습니다`
- "배울 수 있는 시간이었습니다" → "배웠습니다"
- "고민할 수 있는 시간이었습니다" → "고민했습니다"

### SW-08-H `~같은 경우에는`
- "저 같은 경우에는" → "저는"
- "일본 같은 경우에는" → "일본은"

### SW-08-I `했으면 좋겠다` 남용
- 의지 표현 → "하겠습니다", "하고 싶습니다"
- 권유 → "하십시오", "합시다"
- "~할 수 있었으면 좋겠습니다" → "~하겠습니다" / "~하기 바랍니다"

## SW-09: 수동태 남용

### SW-09-A `되어지다` → `되다` 또는 능동
- "배치되어진" → "배치된" → "배치한"

### SW-09-B `~이 요구된다` → 능동
- "지원이 요구된다" → "지원해야 한다"
- "전문성이 요구되어진다" → "전문성을 갖추어야 한다"

### SW-09-C `~을 필요로 한다` → 형용사형
- "지원을 필요로 한다" → "지원이 필요하다"

### SW-09-D `맡겨진`·`주어진` → 능동
- "맡겨진 일" → "맡은 일"
- "주어진 기회" → "받은 기회" / "생긴 기회"

### SW-09-E `보여진다` → `보인다`

## SW-10: 중복·포함 오류

| 원문 | 수정 |
|---|---|
| 수급받다 | 수급하다 / 급여를 받다 |
| 결연 체결 | 결연하다 |
| 접수받다 | 접수하다 / 받다 |
| 교육시키다 | 교육하다 |
| 매년마다 | 매년 |
| 과반수 이상 | 과반수 |
| 가능할 수 있다 | 가능하다 |
| ~이후부터 | ~이후 |
| 제공해 주다 | 제공하다 |
| 함께 동행하다 | 동행하다 |

## SW-11: 띄어쓰기

### SW-11-A 보조용언
- 원칙: 띄어 씀. "해 보다", "해 주다", "잡아 드리다"
- 예외 허용(붙여쓰기): 관용적 고정 표현에 한함.

### SW-11-B `~ㄴ 데`
- 뒤에 조사 "에"를 붙여 자연스러우면 → 띄어 씀.
- "높이는 데 도움이 된다" ✓ / "높이는데 힘들다" (접속어미는 붙임)

### SW-11-C `함께하다`
- 한 단어(함께 행동하다) → "함께하다" (붙임)
- 부사+하다(따로 강조) → "함께 했다" (띄움)

### SW-11-D 파생접미사 `받다`
- "감동받다", "후원받다", "칭찬받다" → 붙여 씀.

---

## SW-12: 어휘 혼동

### SW-12-A `틀리다` vs `다르다`
- 옳고 그름 → "틀리다". 같지 않음 → "다르다".

### SW-12-B `전하다` 오용
- 직접 건네줄 때 → "주다", "드리다", "나누다"
- 중간 매개로 전달할 때 → "전하다"

### SW-12-C `접수하다` 방향
- 신청자가 제출할 때 → "제출하다", "내다"
- 기관이 받을 때 → "접수하다"

### SW-12-D `고맙다` vs `감사하다`
- 마음 상태 → "고맙다" (형용사)
- 감사 행위 → "감사하다" (동사)
- "감사한 말씀" → "고마운 말씀"

### SW-12-E `너무` 긍정 오용
- 긍정 맥락 → "정말", "무척", "매우", "참"

### SW-12-F `~다라고`·`~다라는`
- 간접 인용 → "~다고", "~다는"

### SW-12-G `위탁` vs `수탁`
- 복지관은 정부로부터 위탁받음 → "수탁운영"
