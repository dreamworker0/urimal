## 카테고리 1: 호응·짝 오류 (SW-01)

**정의**: 주어·서술어, 부사·동사, 조사 등 문장 성분 간 호응이 맞지 않는 경우.

| 패턴ID | 패턴명 | 심각도 | 예시 (원문 → 수정) |
|---|---|---|---|
| SW-01-A | 주어-술어 불일치 | S1 | "권한과 책임이 너무 클 수도 있습니다" → 맥락에 맞는 술어로 수정 |
| SW-01-B | 부사-동사 불일치 | S2 | "일주일간 시작합니다" → "일주일간의 ~을 시작합니다" |
| SW-01-C | ~부터 ~까지 짝 누락 | S2 | "기획·진행·평가까지" → "기획부터 평가까지" |
| SW-01-D | 높임말 호응 불일치 | S1 | "과장님께서 해주셨습니다. 이야기를?" → "말씀을" |
| SW-01-E | 주어 누락으로 인한 행위자 혼동 | S2 | "고민하다가 추천해주셨습니다" → 주어 명시 |

## 카테고리 2: 논리 오류 (SW-02)

**정의**: 역접·인과·시제 등 논리 관계가 어색하거나 모순인 경우.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-02-A | 역접 오류 (~지만) | S2 | "소박한 잔치였지만 자연스러운 모습입니다" → 소박함과 자연스러움은 반대가 아님 |
| SW-02-B | 인과 오류 | S2 | "예상하기 어렵지만 바라봅니다" → "예상하기 어려우니 바라봅니다" |
| SW-02-C | 내일/오늘 시제 혼동 | S1 | "내일은 어린이날입니다… 오늘만큼은" |
| SW-02-D | 말하고 행동 모순 | S1 | "말해주세요" 직후 "말보다 포옹이 큰 힘" |

## 카테고리 3: 시제·태·구조 통일 오류 (SW-03)

**정의**: 한 글 안에서 시제·경어체·문체가 통일되지 않은 경우.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-03-A | 했습니다/하였습니다 혼용 | S3 | 같은 글에서 두 형태 혼용 |
| SW-03-B | 높임말 통일 안 됨 | S2 | "어르신께서 담당합니다" / "어르신이 담당하십니다" 혼용 |
| SW-03-C | 나열 구조 불균형 | S2 | "어떤 음식을 만들면 좋을지, 언제 할지" → 구조 통일 |

## 카테고리 4: 관용구 오남용 (SW-04)

**정의**: 사회복지 문서에 자주 등장하지만 의미가 부정확하거나 어색한 관용구.

| 패턴ID | 패턴명 | 심각도 | 예시 (원문 → 수정) |
|---|---|---|---|
| SW-04-A | `관계하다` 오남용 | S2 | "이웃과 관계하며" → "이웃과 어울리며", "왕래하며" |
| SW-04-B | `고민하다` 오남용 | S2 | "고민했습니다" → "숙고했습니다", "검토했습니다", "살폈습니다" |
| SW-04-C | `이야기 나누다` 남발 | S3 | → "토의했습니다", "의논했습니다", "상담했습니다" |
| SW-04-D | `물론` 오용 | S2 | 당연하지 않은 것에 "물론" 사용 |
| SW-04-E | `가지다`·`갖다` 명사화 오용 | S2 | "시간을 가졌습니다" → "~했습니다", "모임을 갖는다" → "모인다" |
| SW-04-F | `대하여`·`대해서` 과잉 | S3 | "필요성에 대해서는 원하고" → "필요성은" |
| SW-04-G | `에 있어서` 과잉 | S2 | "실천에 있어서" → "실천에서" |

## 카테고리 5: `해 주다` 과잉 (SW-05)

**정의**: 당사자가 스스로 하는 일인데 "~해 주다"로 표현하여 주체가 사회사업가인지 당사자인지 혼동되는 경우.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-05-A | 당사자 행위에 `~해 주다` | S2 | "참여해 주셨습니다" → "참여하셨습니다" |
| SW-05-B | 당사자 자발 행위 혼동 | S1 | "추석 잔치를 이뤄주셔서" → 누가 한 일인지 불분명 |

## 카테고리 6: 사업화 문체 (SW-06)

**정의**: 자연스러운 행위를 명사+하다 형태로 과도하게 사업 용어화한 경우.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-06-A | 명사+활동/하다 과잉 | S2 | "독서활동을 했다" → "책을 읽었다", "요리활동을 했다" → "요리했다" |
| SW-06-B | `나눔`+명사 나열 | S2 | "사례 나눔을 하면" → "사례를 나누면" |
| SW-06-C | 용언 쪼개기 | S2 | "접수를 받고 있습니다" → "접수하고 있습니다", "걱정이 되다" → "걱정되다" |

## 카테고리 8: 모호한 표현 (SW-08)

**정의**: 뜻이 명확하지 않거나 책임을 회피하는 듯한 우회적 표현.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-08-A | `~것 같아요` 남용 | S2 | "좋은 것 같아요" → "좋아요" (자기 느낌은 직접 말하기) |
| SW-08-B | `부분` 남발 | S2 | "걱정되는 부분" → "걱정되는 점", "걱정되는 것" |
| SW-08-C | `지점` 남발 | S2 | "함께할 수 있는 지점" → "함께할 수 있는 일" |
| SW-08-D | `~도록 하겠습니다` | S2 | "가도록 하겠습니다" → "가겠습니다" |
| SW-08-E | `~기로 하겠습니다` | S2 | "마치기로 하겠습니다" → "마치겠습니다" |
| SW-08-F | `~라는 생각이 들었습니다` | S2 | → "~라고 생각했습니다" 또는 생각 내용 직접 진술 |
| SW-08-G | `~할 수 있는 시간이었습니다` | S1 | "배울 수 있는 시간이었습니다" → "배웠습니다" |
| SW-08-H | `~같은 경우에는` | S2 | "저 같은 경우에는" → "저는" |
| SW-08-I | `했으면 좋겠다` 남용 | S3 | "가겠습니다"·"가십시오" 등 직접 표현으로 대체 |

## 카테고리 12: 어휘 혼동 (SW-12)

**정의**: 뜻이 비슷하지만 다른 어휘를 혼동하여 쓴 경우.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-12-A | `틀리다` vs `다르다` | S2 | "성격이 틀리다" → "성격이 다르다" |
| SW-12-B | `전하다` 오용 | S2 | "송편을 전하고 싶으세요?" → "주고 싶으세요?" |
| SW-12-C | `접수하다` 방향 오류 | S2 | "신청서를 접수하다"(신청자 입장) → "제출하다" |
| SW-12-D | `고맙다` vs `감사하다` | S3 | "감사한 말씀" → "고마운 말씀" |
| SW-12-E | `너무` 긍정 오용 | S2 | "너무 멋있어요" → "정말 멋있어요" |
| SW-12-F | `~다라고`·`~다라는` | S2 | 간접 인용에 직접 인용 어미 사용 |
| SW-12-G | `위탁` vs `수탁` | S1 | "복지관이 위탁운영" → "수탁운영" |

## 카테고리 13: 일반 맞춤법 및 순화어 (SW-13)

**정의**: 기본적인 맞춤법을 어겼거나, 번역투 문장, 어려운 한자어, 일본어 잔재 등을 사용한 경우.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-13-A | 잦은 맞춤법 오류 | S1 | 든지/던지, 로서/로써, 안/않, 되/돼, 데/대 혼동 |
| SW-13-B | 불필요한 외래어 | S2 | "니즈" → "욕구/필요", "케어" → "돌봄", "인프라" → "기반시설" |
| SW-13-C | 어려운 한자어 / 행정 용어 | S2 | "익일/명일" → "다음 날/내일", "별첨" → "붙임", "통보" → "알림", "상기" → "위" |
| SW-13-D | 과도한 줄임말 | S3 | "사복" → "사회복지사", "프그램" → "프로그램" |
| SW-13-E | `~의` 남용 (일본어투) | S2 | "나의 살던 고향" → "내가 살던 고향", "문제의 해결" → "문제 해결" |
| SW-13-F | 번역투 문장 | S2 | "~에 위치해 있다" → "~에 있다", "~에 다름 아니다" → "~와 같다" |
| SW-13-G | 불필요한 복수형 표기 | S3 | "저희들" → "저희", "많은 사람들이" → "많은 사람이" |
| SW-13-H | 사물 존칭 오류 | S1 | "문의하신 일정은 안되십니다" → "안됩니다", "커피 나오셨습니다" |

## 카테고리 14: 차별적·권위적·시혜적 표현 (SW-14)

**정의**: 특정 집단에 대한 편견, 차별적 어휘, 사회복지 제공자의 권위적·시혜적 태도가 드러나는 표현.

| 패턴ID | 패턴명 | 심각도 | 예시 |
|---|---|---|---|
| SW-14-A | 장애인 차별·비하 표현 | S1 | "정상인" → "비장애인", "장애우/장애자" → "장애인", "절름발이 행정", "눈먼 돈" |
| SW-14-B | 가족 형태 차별 표현 | S1 | "결손가족/편부모" → "한부모가족/조손가족", "미혼모" → "비혼모" |
| SW-14-C | 성차별적 표현 | S1 | "여의사/여직원" → "의사/직원", "유모차" → "유아차", "저출산" → "저출생" |
| SW-14-D | 시혜적·권위적 묘사 | S1 | "불우이웃" → "도움이 필요한 이웃", "수혜자" → "이용자/참여자", "베풀다" |
| SW-14-E | 직업 비하 표현 | S1 | "청소부" → "환경미화원", "파출부" → "가사도우미" |
| SW-14-F | 나이/세대 차별 표현 | S1 | "노인네", "할망구", "요즘 애들은" 등 세대 비하적 뉘앙스 |
| SW-14-G | 인종/지역 차별 표현 | S1 | "혼혈" → "다문화", "살색" → "살구색" |

## 심각도 기준 (재확인)

- **S1 결정적**: 한 번만 나와도 명백한 오류. 무조건 수정.
- **S2 강함**: 맥락에 따라 허용되기도 하나, 반복 시 반드시 수정.
- **S3 약함**: 다른 패턴과 중첩되거나 글의 수준을 낮출 때 수정.
